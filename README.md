# Klix

Klix is a system for installing, configuring, and upgrading the operating system and printing software on Klipper-based 3D printers.

It's a NixOS installation, but you don't need to know anything about NixOS to use Klix.

## Getting started

To use Klix, visit [the website](https://klix.code.supply/) and follow the instructions.

Read [the documentation](https://klix.code.supply/docs) to understand more about what Klix does.

Please file any issues here, under the Issues tab.
